# Todo List Hackathon Project

A modern full-stack task management application with AI chatbot integration, advanced features, and cloud-native architecture.

![Status](https://img.shields.io/badge/status-All_Phases_Complete-success)

## Project Overview

This is a hackathon project demonstrating modern full-stack development practices including:
- **Frontend**: Next.js 16 with React 19, TypeScript, Tailwind CSS 4, shadcn/ui
- **Backend**: FastAPI (Python) with PostgreSQL, AI chatbot integration
- **Infrastructure**: Kubernetes deployment with Minikube and Helm
- **Architecture**: Event-driven with Kafka and Dapr integration

## Quick Start

### Frontend Development

```bash
my Login page code. The left dark-blue panel is very disorganized. Specifically, the text is getting cut off on the left, the icons are not properly aligned with the text, and there is no proper breathing space (padding).

Please fix my code with these instructions:

Fix Padding & Margins: Add a generous padding (e.g., padding: 3rem or p-12 in Tailwind) so the text doesn't touch the screen edges.

Vertical Centering: Use Flexbox to center the content vertically so it doesn't look like it's falling off the top or bottom.

Icon & Text Alignment: Ensure the 'Nexora' logo and the feature list (AI prioritization, Real-time sync) are perfectly aligned horizontally. The icons should be on the left of the text with a consistent gap.

Typography Polish: Make the 'Nexora | Welcome back' heading stand out. Increase the line-height for the description text so it's easier to read.

Modern Styling: Add a subtle radial gradient to the background instead of a flat dark color. Give the icon containers a slight 'glassmorphism' effect (semi-transparent background with a blur).
```

Frontend runs on http://localhost:3000

### Backend Development

```bash
cd backend
uv sync
uv run uvicorn api.main:app --reload
```

Backend runs on http://localhost:8000

### Kubernetes Deployment (Local)

```bash
# Start Minikube
minikube start --cpus=4 --memory=8192 --disk-size=50gb

# Install Redis
helm install redis bitnami/redis --set architecture=standalone

# Build and load images
./k8s/scripts/build-and-load.sh

# Deploy app
helm install todo-app ./k8s/helm/todo-app
```

## Development Workflow

This project uses **Spec-Driven Development (SDD)**:

1. **Specify** (`spec.md`) - Define requirements and acceptance criteria
2. **Plan** (`plan.md`) - Design architecture and components
3. **Tasks** (`tasks.md`) - Break down into actionable tasks
4. **Implement** - Write code following the plan

See `AGENTS.md` for detailed workflow guidelines.
