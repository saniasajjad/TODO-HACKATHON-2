# Evolution of Todo

This project demonstrates the evolution of a Todo application from a simple TUI to a full-stack web application.

## Project Structure

- `phase-1-cli-tui/`: The original Phase I implementation. A localized, in-memory Python TUI application built with Textual.
- `phase-2-web-app/`: The Phase II implementation. A full-stack web application built with Next.js, Neon PostgreSQL, and Better Auth.

## Evolution Journey

1. **Phase I**: Focused on core CRUD logic and a terminal-based UX.
2. **Phase II**: Focuses on persistence, multi-tenancy, and web accessibility.
